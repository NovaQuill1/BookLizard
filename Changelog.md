- A Note About Formatting -
Book Lizard's version number is in the A.B.C format.
A change in A represents a revamp of a part of the app. This could be a full overhaul of the file system or a version for Windows.
A change in B represents a new feature. This could be a more fluid bookmarks system or more themes.
A change in C represents minor changes or bug fixes. This could be a faster launch or more allowed file names.
